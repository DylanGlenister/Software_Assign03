Admin Routes

Administrative functions requiring admin privileges

POST /admin/createAccount

MUST BE LOGGED IN AS ADMIN TO USE
Allows admins to create an account with any role. This includes creating employees or other admins.
The password field would be set as a html password type to hide the characters inputed but for demo purposes just a text field.
