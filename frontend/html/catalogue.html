Product Catalogue Routes

Publicly accessible routes to browse and search products.

GET /catalogue/all

Retrieves a list of all products.

GET /catalogue/search

Searches for products matching a query string (name, description, tags). Allows filtering and sorting.


Response:


Table

Raw JSON Response
GET /catalogue/tagged

Retrieves products associated with specified tags. Allows filtering and sorting.

Products matching ALL selected tags will be shown.

Table

Raw JSON Response
